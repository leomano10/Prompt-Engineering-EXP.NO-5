EXP 5: COMPARATIVE ANALYSIS OF DIFFERENT TYPES OF PROMPTING PATTERNS AND EXPLAIN WITH VARIOUS TEST SCENARIOS

Aim:
To test and compare how different pattern models respond to various prompts (broad or unstructured) versus basic prompts (clearer and more refined) across multiple scenarios. Analyze the quality, accuracy, and depth of the generated responses.

Explanation:

Define the Two Prompt Types:

Write a basic Prompt: Clear, detailed, and structured prompts that give specific instructions or context to guide the model. Based on that pattern type refined the prompt and submit that with AI tool. Get the ouput and write the report.

Prepare Multiple Test Scenarios: Select various scenarios such as: Generating a creative story. Answering a factual question. Summarizing an article or concept. Providing advice or recommendations. or Any other test scenario. For each scenario, create both a naïve and a basic prompt. Ensure each pair of prompts targets the same task but with different levels of structure. Run Experiments with ChatGPT: Input the naïve prompt for each scenario and record the generated response. Then input the corresponding basic prompt and capture that response. Repeat this process for all selected scenarios to gather a full set of results. Evaluate Responses : Compare how ChatGPT performs when given naïve versus basic prompts and analyze the output based on Quality,Accuracy and Depth. Also analyse does ChatGPT consistently provide better results with basic prompts? Are there scenarios where naïve prompts work equally well? Deliverables: A table comparing ChatGPT's responses to naïve and basic prompts across all scenarios. Analysis of how prompt clarity impacts the quality, accuracy, and depth of ChatGPT’s outputs. Summary of findings with insights on how to structure prompts for optimal results when using ChatGPT.

OUTPUT:

A comparison was made between Naïve Prompts and Basic Prompts across different scenarios such as creative story generation, factual question answering, summarization, and recommendation tasks.
```
| Scenario              | Naïve Prompt Performance                          | Basic Prompt Performance                                                    |
| --------------------- | ------------------------------------------------- | --------------------------------------------------------------------------- |
| Creative Story        | Generated a simple story with limited details.    | Generated a well-structured story with clear characters, setting, and plot. |
| Factual Question      | Provided a brief answer with minimal explanation. | Provided accurate information with detailed explanation and examples.       |
| Summarization         | Produced a short summary that missed key points.  | Produced a concise and complete summary covering important concepts.        |
| Advice/Recommendation | Gave general suggestions.                         | Gave specific, relevant, and well-justified recommendations.                |
```
Analysis:

1. Quality:

   * Basic prompts produced more organized and relevant responses.
   * Naïve prompts often resulted in generic outputs.

2. Accuracy:

   * Basic prompts improved accuracy by providing clear context and instructions.
   * Naïve prompts sometimes led to incomplete or less precise responses.

3. Depth:

   * Basic prompts generated detailed explanations and examples.
   * Naïve prompts provided only surface-level information.

Findings:

* ChatGPT consistently performed better with Basic Prompts.
* Clear instructions helped the model understand the task more effectively.
* Naïve prompts worked reasonably well for simple questions but were less effective for complex tasks.
* Prompt clarity significantly improved the quality, accuracy, and depth of responses.

Conclusion:

The experiment demonstrated that Basic Prompts provide better results than Naïve Prompts. Structured and detailed prompts help ChatGPT generate more accurate, comprehensive, and useful responses. Therefore, users should provide clear context and specific instructions to obtain optimal results from AI models.
